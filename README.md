## 项目介绍

这是一个简单的图床，实现了用户的登录，注册，以及上传图片，查看历史记录等基本功能。
项目使用 React+React-Router+styled-components+Ant Design 来组织页面，使用 Mobx 管理状态，LeanCloud 作为后端保存数据。
